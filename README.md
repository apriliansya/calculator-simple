## 🇬🇧 English

A clean, lightweight web-based calculator built with HTML, CSS, and JavaScript — powered by **Vite** for fast development.

### ✨ Features

- ➕ Basic arithmetic: addition, subtraction, multiplication, division
- 🔢 Decimal number support
- 🔁 **DEL** button to remove the last character
- 🗑️ **C** button to clear the display
- 📐 **%** button for modulo operation
- ⚡ Hot reload via Vite dev server

### 🗂️ Project Structure

```
calculator-simple/
├── index.html      # Main markup
├── style.css       # Calculator styling
├── script.js       # Calculator logic
├── screenshot.png  # UI screenshot
└── package.json    # Project config & Vite
```

### 🚀 Getting Started

#### Prerequisites
- [Node.js](https://nodejs.org/) v18 or later

#### Installation & Run

```bash
# Clone the repository
git clone https://github.com/apriliansya/calculator-simple.git
cd calculator-simple

# Install dependencies
npm install

# Start the dev server
npm run dev
```

Open your browser at **http://localhost:5173/**

#### Production Build

```bash
npm run build    # Build to dist/ folder
npm run preview  # Preview the production build
```
